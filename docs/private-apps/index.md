# Private apps

:::info[Coming soon]
Private apps, installing an app for just yourself or your organization, without
publishing it to the public catalog, are on the roadmap. This section is a
placeholder; the flow will be documented here.
:::

## GIF Studio

Available now: **[GIF Studio](/docs/private-apps/gif-studio)** turns your own PNGs into a
panel-accurate animated GIF, no code and no pull request. Run `gdn gifstudio` and drop
pictures in.

## What to expect

- Build and preview exactly like a public app (same two files, same `gdn` commands).
- Keep it out of the public catalog, no pull request required.
- Install it to your own panels privately.

In the meantime, everything in [Getting started](/docs/getting-started/quickstart) and
the [guides](/docs/guides/your-first-app) applies. A private app is just an app you
don't submit.
